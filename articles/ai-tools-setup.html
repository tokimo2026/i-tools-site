AI副業ツール全体像と選び方の基準

まず最初に言っておきたいのは、ツールは「多く使えばいい」わけじゃないってこと。俺が厳選した基準は3つだ。

  • 無料か低コストで始められる（初期投資ゼロから稼ぎを作る）
  • AI副業の収益化に直結する（趣味ツールじゃなく実戦向き）
  • 組み合わせて使える（ツール同士がシナジーを生む）

以下の図で全体像を把握してほしい。ざっくり言うと、「作る→公開→稼ぐ」の3ステップに対応したツールを選んでいる。

💡 ポイント：全部一気に登録する必要はない。まずClaude Pro + note + CrowdWorksの3つを先に揃えれば、最速で収益化できる。残りは稼ぎながら追加していけばOK。

① Claude Pro｜AI副業の核心エンジン

Claude ProはAnthropicが提供するAIアシスタントの有料プランだ。無料版のClaude.aiもあるが、AI副業で本気で稼ぐなら課金一択。理由は明快で、使用量の制限が大幅に緩和されるから。無料版だと1日の使用上限にすぐ引っかかって仕事にならない。

登録手順

  1. claude.ai にアクセスしてGoogleアカウントまたはメールで無料アカウントを作成
  2. 左下の「Upgrade to Claude Pro」をクリック
  3. 月額$20（約3,000円）をクレジットカードで支払い
  4. 有料プラン有効化後、Claude 3.5 Sonnet / Claude 3 Opusが使い放題になる

初期設定のコツ

登録したらすぐに「プロジェクト」機能を設定しよう。プロジェクトごとにシステムプロンプトを保存できるから、「ライティング用」「コード用」「SNS投稿用」と分けておくと作業効率が3倍くらい上がる。

⚠️ 注意：Claude Proは1ヶ月の無料トライアルはない。ただし、月払いなので試してみて「合わない」と思ったらすぐ解約できる。まずは1ヶ月使い倒してみることをすすめる。

② GitHub｜コードと成果物を管理する

「俺、エンジニアじゃないし…」って思った人、待ってほしい。GitHubはエンジニア専用ツールじゃない。AI副業においては主にポートフォリオとして使うのが目的だ。Claudeに作ってもらったアプリやスクリプトを公開することで、クライアントへの信頼感が爆上がりする。

登録手順

  1. github.com でメールアドレスとユーザー名を設定して無料アカウント作成
  2. メール認証を完了させる
  3. プロフィール画像・自己紹介を日本語で記入（クライアントが見るから重要）
  4. 最初のリポジトリを「Public」で作成してみる

非エンジニアでも使えるGitHub活用法

実際のところ、俺がGitHubで一番使うのはNetlifyとの連携だ。GitHubにHTMLファイルをプッシュするだけで、Netlifyが自動でサイトを更新してくれる。コマンドラインを使わなくてもGitHub Desktopというアプリで全部GUIで操作できるから安心していい。

💡 コツ：READMEファイルにClaudeを使って作ったものの説明を書いておくと、それ自体がポートフォリオになる。「AI×○○」という実績をどんどん積み上げていこう。

③ Netlify｜無料でサイトを公開する

Netlifyは静的サイトを無料でホスティングできるサービスだ。AI副業においてはアフィリエイトサイトやポートフォリオサイトの公開場所として使う。独自ドメインも無料で1つ設定できる（netlify.appのサブドメイン）。

登録手順と初期設定

  1. netlify.com にアクセスし、GitHubアカウントでSign up
  2. 「New site from Git」→ GitHubリポジトリを選択
  3. ビルド設定はデフォルトのままでOK（静的HTMLの場合）
  4. 「Deploy site」をクリックすれば数十秒でサイトが公開される
  5. 「Site settings」からサイト名をカスタマイズ（例：ai-jissen-lab.netlify.app）

Netlifyの無料プランでできること

無料プランでも月間100GBの帯域幅、自動HTTPS、継続的デプロイと十分な機能が使える。月に数万PV程度なら全然余裕だ。俺はこのサイトも最初はNetlifyで始めた。

✅ 実績：このAI実践ラボのサイトも初期はNetlifyの無料プランで運営していた。月3〜5万PVくらいまでなら無料枠で十分対応できる。

④ note｜知識を売るプラットフォーム

noteはAI副業の中でも特に即金性が高いプラットフォームだ。自分の知識・ノウハウを有料記事として販売できる。AI副業の実体験、プロンプト集、ワークフロー解説などを記事にして売ることができる。

登録手順

  1. note.com でメールアドレス登録またはGoogle/Twitterでサインアップ
  2. クリエイター情報（ペンネーム・プロフィール画像・自己紹介）を設定
  3. 「売上振込先」を設定（PayPay銀行または通常の銀行口座）
  4. 初めての記事を無料で書いてみて、反応を確認する

noteで稼ぐための戦略

noteで稼ぐためのポイントは「価格設定を300〜500円に抑える」ことだ。高すぎると売れないし、安すぎると手間に見合わない。俺の経験では500円の記事が一番コンバージョン率が高かった。

⑤ CrowdWorks｜案件を獲得する

CrowdWorksは日本最大のクラウドソーシングプラットフォームだ。AI副業で実際の収益を作るには、ここでライティング・データ入力・Webデザイン案件を受注するのが最速だ。

登録手順

  1. crowdworks.jp でメールアドレス登録
  2. 「ワーカー（仕事を受ける側）」として登録
  3. 本人確認書類をアップロード（収益化に必要）
  4. プロフィールに「AI活用歴」「得意分野」を具体的に記載
  5. 実績ポートフォリオをGitHubまたはnoteのリンクで追加

登録無料。AIライティング案件が豊富。まず無料登録して案件を眺めてみよう。

⑥ Lancers｜もう一つの案件獲得先

LancersはCrowdWorksと並ぶ大手クラウドソーシングだ。単価が高めの案件が多い印象がある。特にライティング・翻訳系の高単価案件を探すならLancersも登録しておいた方がいい。

CrowdWorksとの使い分け

俺の使い方はこうだ。CrowdWorksは量×単価で安定収益を作る場所、Lancersは高単価案件にチャレンジする場所、という感じで使い分けている。両方に登録しておいて、毎朝新着案件をチェックするのが基本ルーティン。

登録手順

  1. lancers.jp でアカウント作成（GoogleまたはLINEでも可）
  2. フリーランサーとして登録
  3. スキルタグに「AIライティング」「ChatGPT活用」を追加
  4. ポートフォリオを最低1件アップロード（noteの記事でもOK）

⑦ A8.net｜アフィリエイトで稼ぐ

A8.netは日本最大のアフィリエイトプログラムだ。このAI実践ラボで俺が実際に使っているアフィリエイト収益の柱の一つ。記事を書きながら収益を積み上げる仕組みを作るのに最適だ。

登録手順

  1. a8.net でメールアドレス登録
  2. サイト情報（URL・カテゴリ・月間PV）を入力（まだサイトがなければ今後作る旨を記載でもOK）
  3. 審査通過後、各広告主にプログラム申請
  4. 承認されたリンクを記事内に貼る

AI副業×アフィリエイトの相性

AI副業ブログ・サイトと相性がいいアフィリエイト案件は、学習サービス・ツール系・クラウドソーシングなどだ。CrowdWorksやLancersもA8.netでアフィリエイト登録できる。自分が使っているサービスを紹介するのが一番説得力があって稼ぎやすい。

⑧ Canva｜デザインを爆速で作る

Canvaはデザイン初心者でもプロ並みのビジュアルを作れるツールだ。AI副業においてnoteのサムネイル・SNSバナー・提案資料を作るのに使う。無料でも十分使えるが、Canva Proになるとさらに作業効率が上がる。

登録手順

  1. canva.com でGoogleアカウントまたはメールでサインアップ
  2. 「何のためにCanvaを使いますか？」→「個人」または「フリーランサー」を選択
  3. テンプレートライブラリをざっと眺めて使えそうなテンプレをブックマーク
  4. まず「note用サムネイル（1280×670px）」のテンプレを見つけてカスタマイズしてみる

Canva × Claude活用術

俺がやっているのはClaudeにサムネイルのキャッチコピーを生成させてから、Canvaで貼り付けるという流れだ。テキストをAIに考えてもらって、ビジュアルをCanvaで仕上げる。この2ステップで10分以内にnote用サムネが完成する。

💡 時短テク：Canvaの「ブランドキット」機能に自分のサイトカラー・フォントを登録しておくと、毎回設定する手間が省ける。統一感のあるビジュアルが量産できるようになる。

登録の推奨順序と初期設定チェックリスト

全部一気に登録しようとすると混乱するから、以下の順序でやるといい。

Week 1：収益の土台を作る

  • ☐ Claude.aiで無料アカウント作成
  • ☐ noteアカウント登録・プロフィール設定
  • ☐ CrowdWorksアカウント登録・本人確認

Week 2：公開基盤を整える

  • ☐ GitHubアカウント作成
  • ☐ Netlifyアカウント作成・GitHub連携
  • ☐ Claude Proへのアップグレード

Week 3：収益チャンネルを増やす

  • ☐ Lancersアカウント登録
  • ☐ A8.netアカウント登録・サイト審査申請
  • ☐ Canvaアカウント登録・ブランドキット設定
⚠️ 重要：各プラットフォームでプロフィールを充実させることが最重要タスク。写真・自己紹介・実績を丁寧に書いた人の方が、圧倒的に案件を獲得しやすくなる。手を抜かないでほしい。
